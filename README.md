# 🎙️ My Own Voice Assistant (Python)

A personal AI voice assistant built with Python that can listen to your voice commands, process requests, and respond using voice output.

The project focuses on speech recognition, automation, and building a smart assistant experience.

---

## 🚀 Features

- 🎤 Voice command recognition
- 🔊 Text-to-speech voice response
- 🌐 Web search support
- ⏰ Time and date information
- 🎵 Open applications and websites
- 🤖 Smart command handling
- 🧠 AI-based response support (optional)
- 🌍 Multi-language support (English / Odia / Hindi)

---

## 🛠️ Technologies Used

- Python
- Speech Recognition
- Text To Speech (TTS)
- PyAudio
- gTTS
- OS Automation
- APIs
